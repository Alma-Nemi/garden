# Garden Monorepo

Modern frontend monorepo setup:

- React 19
- Vite 6
- Storybook 8
- Turborepo
- Biome

## Structure

- apps/web → main app
- apps/storybook → component explorer
- packages/ui → design system
- packages/theme → theme system

## Architecture goal

Stable, interview-ready frontend foundation with reproducible builds.
